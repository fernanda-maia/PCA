## Como rodar a aplicação
1. Abra o terminal e caminhe até a pasta educapp 
~~~shell
cd PCA/app/educapp
~~~
2. Faça a instalação das dependências:

~~~shell
yarn install
~~~
3. Inicie a aplicação:

~~~shell
yarn start
~~~
4. No seu navegador de preferência acesse o **_http://localhost:3000/_**
